# 特性ビルダーでの演算順序 {#order-of-operations-in-trait-builder}

[!UICONTROL Trait Builder]では、以下に示す演算順序（優先順位の高い順）に従って、式を評価します。優先順位が高い演算子で定義された特性要素は、他の優先順位の演算子より先に評価されます。ここでは、優先順位に従って、高い順に各演算子をランク付けしています。
