## March, 2019 {#aam-march-updates}

| トピック | 説明 |
|--- |----|
| [グローバルデータソース](../features/global-data-sources.md) | グローバルデータソースにはすべての Audience Manager ユーザーがアクセスでき、Apple、Samsung、Microsoft、Roku、Android などのデバイスの製造元によって生成されたデバイス広告 ID が含まれています。これらの ID は、製造元が広告目的で利用を許可したものです。グローバルデータソースを使用すると、デバイス ID を同期し、これらのマッピングでキーにされたデータを読み込みまたは書き出しできます。Audience Manager では、ユーザーが読み込んだデバイス広告 ID（DAID）の形式が、デバイスの製造元が指定した標準形式と一致しているかを検証します。 |
| [ID マッピングの上限](../features/administration/usage-limits.md#id-mapping-limits) | デバイス ID に関連付けられたクロスデバイス ID マッピングの数を制限することで、セグメント化、レポートおよびセグメントの書き出しを改善しています。新しく設定された制限により、インジェストおよびダウンストリーム処理時に間違った統合や不正確なデータセットが発生した場合の影響を軽減します。<br/>以下に、新しい制限を示します。 <ul><li>クロスデバイス ID ごとに 100 デバイス ID</li><li>デバイス ID ごとに 10 クロスデバイス ID</li><li>1 デバイス ID に対して 1000 デバイス ID</li></ul> |
