# Audience Manager のタイムゾーン{#time-zones-in-audience-manager}

Audience Manager は UI 全体で協定世界時（UTC）を使用しています。

別途指定しない限り、[!DNL Adobe] [!DNL Audience Manager] UI で選択できる日付および日付範囲はすべて[協定世界時（UTC）](https://www.timeanddate.com/worldclock/timezone/utc)となります。例えば、[送信先の作成](../features/destinations/manage-destinations.md#segment-mappings)のフローでセグメントと送信先をマッピングする際に選択する開始日と終了日は、午前 0 時（UTC）となります。[!DNL Audience Manager] のすべての日付に関して同じことが当てはまります。
